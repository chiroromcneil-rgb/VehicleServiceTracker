# ITS203 Individual Assignment – Progress Report

## Project Title
Vehicle Service Booking and Maintenance Tracker

## Completed So Far
- Created the Windows Forms project structure.
- Added OOP model classes for Person, Owner, Vehicle, Car, ServiceBooking, and MaintenanceRecord.
- Added inheritance between Person and Owner, and between Vehicle and Car.
- Added an abstract Vehicle class and an IServiceReport interface.
- Added file-based CSV storage for vehicles, bookings, and maintenance records.
- Added the main form for viewing vehicles and bookings.
- Added forms for adding/editing vehicles, creating bookings, recording maintenance, and viewing reports.
- Added validation and try-catch exception handling.

## Challenges Encountered
- Designing the project so that it clearly demonstrates all required OOP principles.
- Choosing simple storage that works without extra database setup.
- Making the Windows Forms interface easy to use.
